解决node-sass-called-fsevents-install-error

2017-01-10

背景

Windows环境下运行某个vue全家桶做的项目，git clone 后，运行cnpm install报错，运行npm run dev后也报错。初步判断报错原因是node-sass没有编译通过导致。

报错情况

cnpm install

npm run dev

解决

  • 确认先安装一下最新版node-sass(非必需)
  • 本地编译node-sass

到此为止，应该可以了。
